## CodeIgniter 4 Boilerplate Maintencance (Ordenes de Mantenimiento)
Maintenance Management Boilerplate

This repository provides a robust foundation for creating and managing Maintenance Orders. It is designed to be versatile, allowing users to track service history for computer equipment, vehicles, or any asset requiring regular upkeep.

### Key Features:
* Multi-purpose: Suitable for IT departments, automotive shops, or general repair centers
* Asset Tracking: Register and monitor maintenance for any physical item
* Scalable Architecture: Built as a module that can be integrated into any workshop management system

## Requirements
* PhpCfdi\SatCatalogos

## Installation

### Run commands
	composer require julio101290/boilerplatemaintenance
	
### Run command for migration and seeder

	boilerplatemaintenance:installmaintenance
