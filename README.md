# CodeIgniter 4 Boilerplate Maintenance  
### Maintenance Orders Management Module

A professional and scalable **Maintenance Orders Management module** built for **CodeIgniter 4**.

This boilerplate provides a solid foundation for managing maintenance operations across different industries such as:

- 🖥 IT Departments  
- 🚗 Automotive Workshops  
- 🏭 Industrial Equipment Maintenance  
- 🛠 General Repair Centers  

---

# ✨ Features

✔ Modular architecture (plug & play in existing projects)  
✔ Maintenance orders tracking  
✔ Department catalog management  
✔ Employee management  
✔ Asset assignment per employee  
✔ Dashboard with pending maintenance overview  
✔ Scalable structure for integration with larger ERP systems  

---

# 📦 Requirements

- PHP 8.1+
- CodeIgniter 4
- `phpcfdi/sat-catalogos`
- `julio101290/boilerplatelog`
- `julio101290/boilerplateinventory`

---

# 🚀 Installation

Install via Composer:

```bash
composer require julio101290/boilerplatemaintenance
```

Run migrations and seeders:

```bash
php spark boilerplatemaintenance:installmaintenance
```
